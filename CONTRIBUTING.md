# Contributing
## Commit message guidelines
### Commit title
`[your-username] commit-summary`
### Commit content
Every significant/distinct change in the commit should be listed and tagged like this:  
`[tag] change`
#### Tag examples
* [Added]  
* [Changed]  
* [Fixed]  
* [Removed]
### An example of a commit
```
[Bluudek] Style changes

[Added] New color palette
[Removed] Old colors
```
## Pull request message guidelines
### Pull request title
`[your-username] pull-request-summary`
### Pull request content
```
Description
pull-request-description
Changes
commit-title
commit-content
...
```
### An example of a pull request
```
# Description
Rewrite of win checks and added a new game mode
# Changes
## [[Bluudek] Win checks](link to the commit)
* **[Changed]** Win check logic to a faster algorithm
* **[Fixed]** Redundant logic on player click
## [[Bluudek] New game mode](link to the commit)
* **[Added]** Option to select - player vs bot
* **[Added]** Easy bot logic
* **[Added]** Medium bot logic
* **[Added]** Hard bot logic
```
